Provides flexible and advance news ticker. Display it via shortcode and more.

== Description ==

This plugin help you to view the latest posts or page on your website.This plugin also have three type of animation such as Fade Effects, Slide Effects. You can also control and adjust color and animation **Admin Panel**. You can display it from all Post or specific Page and much more!

= Features Include =

* Three type of animation.
* Display multi-functional data.
* Display from all, specific Post and Page.
* Exclude current News
* Control and adjust multi-color and animation Admin panel.


== Viw by Shortcode ==

1. General Options

<pre>[advance_newsticker_shortcode]</pre>

2. Ticker Form Page, Post and Title

<pre>[advance_newsticker_shortcode title="News" per_page_item="3" post_type="post"]</pre>

3. Ticker Effects Options

**Available Ticker Effects Options** <pre> {Fade, Slide Left, Slide Right, Slide Down, Slide Up }</pre>

<pre>[advance_newsticker_shortcode effect_type="fade"]</pre>


== Installation ==

**Through Dashboard**

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **Advance News Ticker** in the search box and click on search button.
3. Find Advance News Ticker plugin.
4. Then click on Install Now after that activate the plugin.
5. Go to the Menu page **Advance News Ticker -> News Ticker Options Settings**.
6. Find **Advance News Ticker** News Ticker Options Settings.

	__<?php echo do_shortcode('[advance_newsticker_shortcode]');__


**Installing Via FTP**

1. Download the plugin to your hardisk.
2. Unzip.
3. Upload the **Advance News Ticker** folder into your plugins directory.
4. Log in to your WordPress admin panel and click the Plugins menu.
5. Then activate the plugin.
6. Go to the menu page**-> Advance News Ticker**.
7. Find **Advance News Ticker** News Ticker Options Settings.
